# Visual Concepts

These SVGs are article-first visuals for the offshore-dollar observability argument.

## Files

- `01_visibility_matrix.svg`: compares what old offshore dollars hide vs what stablecoins reveal.
- `02_public_dollar_topology.svg`: shows Ethereum mainnet and L2s as an observable dollar topology.
- `03_observable_signals_dashboard.svg`: frames stablecoins as a public instrument panel for offshore-dollar behavior.

## Core Visual Thesis

The strongest version of the argument is not that stablecoins make all offshore-dollar activity transparent. They do not reveal user identity or exact geography.

The stronger, defensible claim is:

> Stablecoins make a new slice of offshore-dollar circulation publicly observable: flow, timing, routing, concentration, chain/layer structure, and smart-contract interaction.

## How To Use

Use `01_visibility_matrix.svg` near the conceptual setup, before Dune data.

Use `02_public_dollar_topology.svg` after introducing the Dune results. It connects the observed metrics to the mainnet/L2 structure.

Use `03_observable_signals_dashboard.svg` as the article's "why this matters" visual: offshore dollars used to be inferred from stress; stablecoins can be queried as live public signals.
